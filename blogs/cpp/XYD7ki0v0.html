    read -p "first number: " firstnu
read -p "second number: " secnu

    # 默认变量

    脚本里可以使用$0``$1等变量，第一个是脚本路径（调用的时候输入的内容，如果只输入文件名的话就只有文件名），后面是输入的参数

    /path/to/scriptname opt1 opt2 opt3 opt4
$0                                $1    $2     $3    $4
    1
    2

    $#:表示参数个数。上面例子是4 $@:表示参数列表。 $*:表示所有参数合成的字符串。

    # case

    基本语法

    case $变量名 in
    "xxxxx" )
        do something
        ;;
    "xxxxx" )
        do something
        ;;
    *) # 最后用*表示剩下的所有值
        do something
        ;;
esac

    # 执行shell脚本

    sh [-nvx] scripts.sh
选项与参数：
-n ：不要执行 script，仅查询语法的问题；
-v ：执行 sccript 前，先将 scripts 的内容输出到屏幕上；
-x ：将执行的 script 内容一步一步显示到屏幕上
